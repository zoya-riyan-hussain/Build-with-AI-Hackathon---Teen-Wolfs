# 🧠 AI Career Intelligence Platform  

🔗 Live App: https://dream-scout-ai.lovable.app/  

---

## 🚀 Overview  
The AI Career Intelligence Platform is an AI-powered web application that helps students discover suitable career paths based on their skills, interests, and goals.  

It provides intelligent recommendations, simulations, and performance insights to help users make informed career decisions.  

> “Don’t just choose a career — experience it before you decide.”

---

## ✨ Features  

### 🔐 Authentication  
- Secure Signup & Login  
- JWT-based authentication  
- Form validation  

### 📊 Dashboard  
- Personalized welcome message  
- Top career match with score (%)  
- Performance trends  
- Quick navigation  

### 🧠 Career Recommendations  
- AI-generated top 3–5 career paths  
- Career Fit Score (%)  
- Matched vs Missing Skills  
- Salary range & growth insights  
- Explanation for each recommendation  

### 🎮 Simulations  
- Quiz and coding challenges  
- Timer-based system  
- Difficulty levels  
- Instant feedback  

### 📈 Performance Tracking  
- Topic-wise performance charts  
- Activity history  
- Time tracking  
- Score trends  

### 🤖 AI Feedback  
- Strengths & weaknesses analysis  
- Personalized improvement roadmap  
- Smart suggestions  

### 💬 AI Chatbot  
- Career guidance  
- Skill recommendations  
- Answers user queries  
- Handles new career requests  

### 🧑‍💼 Admin Panel  
- Manage users  
- Add/edit career paths  
- Approve new career requests  
- Update skills and resources  

---

## 🧠 AI Capabilities  
- Handles incomplete or vague inputs  
- Uses keyword matching + similarity logic  
- Dynamic scoring system (Career Fit %)  
- Provides fallback suggestions  

---

## 🎨 UI/UX  
- Modern SaaS design  
- Responsive (mobile + desktop)  
- Clean layout with smooth navigation  
- Dark/light mode ready  

---

## ⚙️ Tech Stack  

- Frontend: React  
- Backend: Node.js + Express  
- Database: MongoDB  
- Charts: Recharts  
- Authentication: JWT  

---

## 🔁 Workflow  

1. User signs up / logs in  
2. Completes profile (skills, interests, goals)  
3. AI generates career recommendations  
4. User explores simulations  
5. Performance updates  
6. AI feedback adapts dynamically  

---

## 🧪 Error Handling  
- Handles empty or incomplete inputs  
- Provides fallback recommendations  
- Prevents crashes  
- Displays loading and validation states

To build a scalable, AI-driven platform that helps users make better career decisions through intelligent insights and real-time feedback.
